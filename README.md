---

## Overview
This is an exploratory tool for **early health technology assessment (HTA) triage** of digital health, medtech, diagnostics, and service innovations.  
It turns early HTA reasoning (problem → pathway → value → uncertainty → evidence plan) into a short, structured workflow with scoring and visual feedback.

The repository contains a **Streamlit-based prototype** that supports structured self-assessment, prioritised next steps, and exportable outputs.

---

## What it does
The app guides users through key early HTA workstreams (scored on a **0–3 maturity scale**):

- Problem & decision context (PICO-style framing)
- Care pathway placement and workflow fit
- Comparator / current practice
- Stakeholder and adoption context
- Value proposition (clinical, operational, economic)
- Early performance / feasibility signal
- Clinical evidence maturity (trial / service evaluation / RWE)
- Early economic logic (scenario / threshold thinking)
- Key uncertainties and evidence priorities

Outputs include:
- **Weighted readiness score**
- **Radar chart** and **bar chart**
- **Prioritised “next steps”** list
- **CSV export** of scores and weights

---

## Intended use
This tool is designed for **early-stage evaluation and discussion**, particularly in:
- innovation accelerators and venture builders
- translational programmes (e.g., university innovation hubs, AHSN/ICS projects)
- early medtech and digital health teams

It supports **decision framing and evidence planning**, not formal assessment.
